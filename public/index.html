Integração VTLog

VTLog API + Discord

Área para sincronizar entregas pela API oficial do VTLog e manter o Discord como apoio/fallback.

VTLog API direta

A integração direta pode rodar em modo pessoal usando apenas seu VTLOG_STEAM_ID, ou em modo empresa usando VTLOG_API_TOKEN + VTLOG_VTC_ID.

O que a sincronização busca

No modo pessoal, busca seus jobs pelo Steam ID. No modo empresa, busca jobs, membros, estatísticas, caminhões, garagens e entregas em andamento da VTC. Os dados ficam salvos em data/vtlog-api-cache.json.

Webhook direto do VTLog

Se o VTLog permitir configurar Job Webhook, use a URL pública do seu site apontando para:

/api/vtlog/webhook/job

Localmente, teste primeiro com POST /api/vtlog/webhook/test. Para o VTLog chamar seu computador, use ngrok ou publique no Render.
